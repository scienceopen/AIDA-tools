date2juldate

PURPOSE

DATE2JULIANDATE calculates the julian date at 0h UT

SYNOPSIS

function [jd] = date2juldate(date)

DESCRIPTION 

 DATE2JULIANDATE calculates the julian date at 0h UT

 Calling:
 [jd] = date2juldate(date)

 It takes input date as an array with date(1) <-> the year ( 95 -> 1995 )
                                      date(2) <-> month
                                      date(3) <-> day nr in month
