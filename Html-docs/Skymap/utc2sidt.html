utc2sidt

PURPOSE

UTC2SIDT calculates the sidereal time.

SYNOPSIS

function [sidtime] = utc2sidt(date,utc)

DESCRIPTION 

 UTC2SIDT calculates the sidereal time.

 Calling:
 [sidtime] = utc2sidt(date,utc)

 Takes date and utc as argument.
 date(1) - the year ( 1995 - 95 )
 date(2) - month
 date(3) - day nr in month
 utc(1)  - hh
 utc(2)  - mm
 utc(3)  - ss.ms
