point_on_line

PURPOSE

POINT_ON_LINE calculates the vector to a point R_OUT

SYNOPSIS

function [r_out] = point_on_line(r_0,e_l,l)

DESCRIPTION 

 POINT_ON_LINE calculates the vector to a point R_OUT
 that are L away from R_0 in the direction E_L.
 
 Calling:
