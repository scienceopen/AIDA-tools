points_on_line

PURPOSE

POINTS_ON_LINE calculates the vector to a set of points R_OUT

SYNOPSIS

function [r_out] = points_on_line(r_0,e_l,l)

DESCRIPTION 

 POINTS_ON_LINE calculates the vector to a set of points R_OUT
 that are L away from R_0 in the direction E_L.
 
 Example:
   r_0 = [0 0 1];
   e_l = [.1 .8 .3]'/sum([.1 .8 .3].^2).^.5;
   l = [-3:3];
   [r_out] = points_on_line(r_0,e_l,l);
