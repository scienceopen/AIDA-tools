AIDA: Auroral Image Data Analysis tools

Advanced image processing tools and friends
This is the on-line documentation of the AIDA toolbox. It provides a small and self-contained set of functions for single (keograms et al.) and multi-viewpoint image data analysis (tomography, stereoscopic triangulations). Further it includes tools for camera calibration using images of background stars. This fundamental functionality is based on general camera models, and a small ephemeris program.

Initially these functions were designed to work with data from the Auroral Large Imaging System (ALIS) but they should work with any image data. However the set of functions for producing and refining ALIS experiment set-up programs and a set of functions for browsing ALIS data are included as well. These functions can easily be customized and extended for use with other imaging systems. In all the following matlab toolboxes are included:

  • ALIS - functions directly related to ALIS. Contains data browser, experiment-programmer.
    ( intro, examples, function list, m-src-doc)
  • ASK - functions directly related to analysis of data from the Auroral Structure and Kinetics instrument - ASK.
    ( intro, examples, function list, m-src-doc)
  • Camera - Black-box-model of the imaging process. Everything related to camera models, image projections, stereoscopy, camera related image corrections.
    (intro, examples, function list, m-src-doc)
  • Fits_tools - functions for reading and writing fits files, and related image correction functions.
    ( intro, examples, function list, m-src-doc)
  • Imgtools - general image analysis functions.
    ( intro, example, function list, m-src-doc)
  • Tomography - toolbox for doing 3-D cone beam tomographic inversion from a small number of view-points.
    ( intro, examples, function list, m-src-doc)
  • Starcal - program for doing geometric camera calibrations.
    (intro, examples, function list, m-src-doc)
  • Skymap - a star chart, ephemeris program.
    ( intro, examples, function list, m-src-doc)

The following toolboxes are utility-tools, helper functions and building blocks the functions above use:

  • EARTH - WGS-84 geoid model.
    ( intro, examples, function list, m-src-doc)
  • Geometry - some small functions for doing 3-D geometry.
    (intro, examples, function list, m-src-doc)
  • I3D - functions for three-dimensional intensity distributions.
    ( intro, examples, function list, m-src-doc)
  • Inversion - small set of SVD based least square functions.
    ( intro, examples, function list, m-src-doc)
  • tools - Assorted and auxiliary tools with varied origins.
    ( intro, examples, function list, m-src-doc)
