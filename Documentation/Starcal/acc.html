The .acc format

ACC - ALIS Camera Calibration is a format where some necessary information is stored. The format is:

StNr Az Ze startdate stopdate cameramodel optpar 0
The Station number (StNr), azimuth (Az) and zenith (Ze) are taken from the initial information on the image and is intended to be used as lookup-keys, startdate and stopdate are the julian dates for which this optical parameters are to be valid.

If no reliable values can be given for Az and Ze there is good reason to see to that this is obtained.
