 INTERFERENCE_RED - Interference reduction
   
 Calling:

 Input:
   IMG_IN - 2-D double array from which interference is to be
            removed.
   LINES - Line-nr array [(1xN) or (Nx1)] for lines in the
           fft-plane to be removed
