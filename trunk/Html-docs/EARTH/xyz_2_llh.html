xyz_2_llh

PURPOSE

XYZ_2_LLH transforms X,Y,Z in an lat0,long0 centred horizontal system

SYNOPSIS

function [long,lat,h] = xyz_2_llh(lat0,long0,h0,r)

DESCRIPTION 

 XYZ_2_LLH transforms X,Y,Z in an lat0,long0 centred horizontal system
 to ( LONG, LAT, H ).
    
 Calling:
 [long,lat,h] = xyz_2_llh(lat0,long0,h0,r)
 
 See also LATLONGH_2_R
