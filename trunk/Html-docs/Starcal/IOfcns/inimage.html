inimage

PURPOSE

INIMAGE tests if a point (X,Y) is within the image,

SYNOPSIS

function [true] = inimage(x,y,bx,by)

DESCRIPTION 

 INIMAGE  tests if a point (X,Y) is within the image,
 that is within  the ranges: 1 <= X <= BX  and 1 <= Y <= BY
