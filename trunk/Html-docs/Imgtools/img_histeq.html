 img_histeq - histogram equalisation. 
    IMG_IN should be a double array. When HIST_LIM is an integer
    IMG_HISTEQ makes IMG_IN to have a uniform histogram
   
   J = IMG_HISTEQ(I,N) transforms the double array I, returning
